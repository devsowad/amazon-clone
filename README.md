## Amazon clone using NextJS, MongoDB, Redis, Material UI and Stripe

### Run this project

In the project directory, you can run:

```shell
yarn
```

```shell
cp .env.example .env
```

```shell
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
